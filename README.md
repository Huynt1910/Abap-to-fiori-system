# ABAP to Fiori Modernization Cockpit

## Overview

This project analyzes legacy ABAP reports and generates rule-based modernization recommendations for SAP Fiori migration.

## Technologies

- SAP UI5
- OpenUI5
- ABAP
- OData
- GitHub

## Features

- ABAP report analysis
- CDS recommendation
- RAP recommendation
- UI5 migration suggestion
- Dashboard reporting

## Run Project

```bash
npm install
npm start
```
